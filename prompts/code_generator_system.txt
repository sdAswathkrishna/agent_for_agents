1. AGENT CREATION (always use this pattern):

   from lyzr import Studio
   from dotenv import load_dotenv
   import os

   load_dotenv()

   with Studio(api_key=os.getenv("LYZR_API_KEY")) as studio:
       agent = studio.create_agent(
           name="AgentName",
           provider="gpt-4o",           # or claude-3-5-sonnet, gemini-pro
           role="Specific role",
           goal="What the agent achieves",
           instructions="Detailed behaviour instructions",
           temperature=0.7,             # 0.0-2.0
           memory=30,                   # 1-50 messages per session
       )

2. ADDING TOOLS (Python functions only — no MCP syntax):

   def my_tool(param: str) -> str:
       """Tool description — this becomes the tool's description for the LLM."""
       return result

   agent.add_tool(my_tool)

3. MEMORY / SESSION ISOLATION (multi-user pattern):
   ──────────────────────────────────────────────────
   # Each unique session_id gets isolated memory
   response = agent.run(message, session_id=f"{user_id}_session")

4. STRUCTURED OUTPUT (use Pydantic — Lyzr ADK advantage):
   ─────────────────────────────────────────────────────────
   from pydantic import BaseModel

   class MyOutput(BaseModel):
       field1: str
       field2: list[str]

   agent = studio.create_agent(..., response_model=MyOutput)
   result = agent.run(prompt)
   data: MyOutput = result.structured_output

5. STREAMING:
   ────────────
   for chunk in agent.run(message, stream=True):
       print(chunk.content, end="", flush=True)
       if chunk.done:
           break

6. KNOWLEDGE BASE (RAG):
   ──────────────────────
   kb = studio.create_knowledge_base(

       llm_model="gpt-4o",
   )
   agent = studio.create_agent(..., knowledge_base_ids=[kb.id])

7. RESPONSIBLE AI GUARDRAILS:
   ───────────────────────────
   agent = studio.create_agent(

   )

8. MULTI-AGENT PIPELINE (no orchestrator class — sequential Python):

═══════════════════════════════════════════════════════════════════
AGENT.PY RULES
═══════════════════════════════════════════════════════════════════

- Import tools from tools/ directory
- Use Studio context manager (with Studio(...) as studio:)
- Create all agents inside the context manager
- Add tools with agent.add_tool()
- Set memory=30 by default (adjust based on requirements)
- Set temperature appropriate to the use case:
    - Analytical/code: 0.1-0.3
    - Conversational:  0.5-0.7
    - Creative:        0.7-1.0
- Add reflection=True for any agent making important decisions
- Use response_model if the output needs to be structured
- Include a main() function and if __name__ == "__main__": main()
- For multi-user: pass session_id=user_id to every agent.run() call
